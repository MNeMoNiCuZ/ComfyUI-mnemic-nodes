# ✏️ Literal String

A simple string literal input node.

-   **Description**: Provides a multiline text input field. Use this to create a text source in your workflow.
-   **Inputs**:
    -   `value`: Multiline text input field.
-   **Outputs**:
    -   `value`: The string value.

**Features**:
- Supports multiple lines of text
- Great for entering prompts, descriptions, or any text content
