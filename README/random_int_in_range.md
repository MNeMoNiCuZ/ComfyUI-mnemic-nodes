Generates a random integer within a specified range.

-   **Description**: Generates a random integer between min and max values (inclusive). Supports optional seed for reproducibility.
-   **Inputs**:
    -   `min_value`: Minimum value for the random integer (inclusive).
    -   `max_value`: Maximum value for the random integer (inclusive).
    -   `seed` (optional): Seed for random number generator. Use -1 for random results, or set a specific value for reproducibility.
-   **Outputs**:
    -   `random_int`: The generated random integer.

**Note**: If min_value is greater than max_value, they are automatically swapped.
