Generates a random boolean value with optional weighted probability.

-   **Description**: Generates True or False randomly. Adjust probability to weight the randomness.
-   **Inputs**:
    -   `true_probability`: Probability of returning True (0.0 = always False, 1.0 = always True, 0.5 = 50/50).
    -   `seed` (optional): Seed for random number generator. Use -1 for random results, or set a specific value for reproducibility.
-   **Outputs**:
    -   `random_bool`: The generated boolean value.
